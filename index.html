HEXO-搭建个人博客（一）| 搭建基础环境

安装Git

官网下载：https://git-scm.com/

我用的版本是：2.33.0.windows.2

安装Node.js

官网下载：http://nodejs.cn/

我用的版本是：v16.15.0

安装Hexo

在cmd窗口输入指令

npm install -g hexo-cli

在磁盘创建一个文件夹“myblog”，在这个文件夹下右键git bash打开

初始化hexo

hexo init

启动hexo

hexo s

根据提示复制网址打开

http://localhost:4000

将Hexo部署到GitHub

在GitHub上创建个人仓库

生成SSH添加到GitHub上

在cmd窗口输入

接着在cmd窗口中输入下方的命令，然后敲击三下回车

打开.ssh目录，id_rsa.pub是公钥，另一个私钥，复制公钥中的内容

进入到GitHub，点击setting中的SSH，将密钥粘贴进去，点击“Add SSH Key”

部署到GitHub中

修改根目录的配置文件

在自己的根目录中，打开git，输入命令，为了安装deploy-git

npm install hexo-deployer-git --save

依次执行下面三个命令

现在就可以通过“用户名.github.io“进行访问你自己的博客了

文件夹 解释
node_modules node.js模块
scaffolds 文章模板
source md源码
themes 主题源码

Front-matter

通过hexo新建的文章的开头，存在这么一段

通过---包括起来的内容称之为Front-matter，即前置信息，用于渲染md文档，内容还可以自行添加

配置项 意义
title 网页文章标题
date 文章创建日期
comments 文章评论功能是否启动
tags 文章标签
categories 文章分类
keywords 文章关键字
