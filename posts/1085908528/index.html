解决hexo博客中图片不显示问题

发表于2023-04-14|更新于2023-04-17
|阅读量:

解决办法分为两步

一、修改博客的配置

将博客根目录中的_config.yml配置文件中的post_asset_folder设置为true

修改完配置后，在通过hexo new "文件名"创建博客的时候，会同时创建一个跟文件名相同的文件夹

二、下载插件

在根目录输入命令

npm install https://github.com/CodeFalling/hexo-asset-image --save

此后，在书写博客的时候，需要用到的对应图片放入到该新建的文件夹中即可
